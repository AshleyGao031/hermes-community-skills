# 抽检清单规则

> 每次生成报告时，自动生成 `_抽检清单.md`，包含5个最关键的数字。

---

## 抽检项目

1. **总Revenue（合计行）**
2. **拉新GMV（合计行）**
3. **SMROI（合计行）**
4. **拉新Top1变动**（绝对变动额最大的项目组）
5. **Q1达成率**（Q1-B标总计行）

## 核对方法

每个数字标注核对方法（在Excel哪里找）。
用户30秒打勾确认，全部✅则报告可用。

## 输出格式

```markdown
# 抽检清单 — W{X}

| # | 指标 | 报告值 | Excel位置 | ✅/❌ |
|---|------|-------|-----------|-------|
| 1 | 总Revenue | {值} | 周营收-财务口径 → 合计行 → 总营收列 | |
| 2 | 拉新GMV | {值} | 周营收-财务口径 → 合计行 → 拉新GMV列 | |
| 3 | SMROI | {值} | 周营收-财务口径 → 合计行 → SMROI列 | |
| 4 | {项目组}拉新变动 | {值} | 本周-上周原始值计算 | |
| 5 | Q1达成率 | {值} | Q1季度-B标 → 总计行 → 进度%列 | |
```
