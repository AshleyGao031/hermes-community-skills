# 指标定义与数据精度

> 本文件是指标定义、计算公式、精度要求、状态标注规则的唯一来源。

---

## 一、指标定义与计算逻辑

| 指标 | 定义 | 计算公式 |
|------|------|---------|
| COR | 履约成本 | — |
| S&M | 销售市场费用 | — |
| RD | 研发费用 | — |
| GA | 管理费用 | — |
| OP | 经营利润 | Revenue - COR - SM - RD - GA - 公共分摊 |
| OP% | 经营利润率 | OP / Revenue |
| 毛利率 | 毛利占比 | (Revenue - COR) / Revenue |
| SMROI-项目组 | 项目组投放效率 | 拉新GMV ÷ SM |

---

## 二、数据精度要求

| 类型 | 精度 |
|------|------|
| 金额（万元） | 保留 1 位小数 |
| 环比百分比 | 保留 1 位小数 |
| SMROI | 保留 2 位小数 |
| 进度百分比 | 保留 1 位小数 |
| 同比百分比 | 直接使用 Excel 中的格式（如"-18.3%"） |

## 三、🟢🟡🔴 状态标注规则

### 达成率标注
- 🟢 达成 ≥ 95%
- 🟡 达成 85% - 95%
- 🔴 达成 < 85%

### SMROI 环比趋势标注（v6.6.0新增）
判断优先级：
1. **SMROI本身正负**：负数直接🔴，不管变化幅度
2. **从正变负**：🔴（恶化）
3. **从负变正（或更接近0）**：🟢（改善）
4. **都是正数**：增加→🟢，减少→🔴
5. **都是负数**：绝对值减少（接近0）→🟢，绝对值增加→🔴

**示例：**
- SMROI 0.00→-0.38：🔴（从0变负）
- SMROI -0.10→+0.05：🟢（从负变正）
- SMROI 1.52→0.52：🔴（都是正，减少了）
- SMROI 27.55→17.14：🔴（都是正，减少了）

### 进度判断（GMV vs 时间进度）
- ✅ 进度 > 时间进度 = 超前
- ⚠️ 差距 < 10pp = 边缘
- 🔴 差距 > 10pp = 落后

## 四、[待BP确认] 机制

- 所有基于推测的业务原因**必须**标注 `[待BP确认]`
- 报告最后必须汇总所有 `[待BP确认]` 项，按重要性排序
- 格式：`[待BP确认] {推测内容}（影响程度：高/中/低）`
- AI不确定的数据来源也要标 `[数据缺失-请补充]`
