# 数据源映射

> 本文件定义每个分析模块的数据来源 Sheet，以及周报表格的具体字段映射。

---

## 一、模块 → Sheet 映射

> ⚠️ **Sheet名称随季度/月份变化**：Q2数据中是`Q2季度-B标`而非`Q1季度-B标`，是`5月-B标`而非`当月-B标`。脚本会自动匹配，但本文件中的示例名称仅供参考。

| 分析模块 | 数据来源Sheet | 备注 |
|---------|-------------|------|
| 模块1-总营收/拉新Top3/续报Top3 | `周营收-财务口径` | SMROI用财务口径 |
| 模块2-素养项目组 | `周营收-团队口径` | SMROI用团队口径 |
| 模块3-GMV进度 | `{季度}季度-B标` + `{月份}-B标` | Sheet名随季度/月份变化 |
| 模块5-同比 | `{季度}累计同比` + `{月份}同比` | Sheet名随季度/月份变化 |
| 环比变动计算 | 本周Excel + 上周Excel的原始值 | ❌不要用环比%列反推 |

## 二、本周速览表格数据映射

### GMV（万）
- 本周实际 = `周营收-财务口径.合计.总营收`
- 上周实际 = 上周Excel.`周营收-财务口径.合计.总营收`
- 环比 = `周营收-财务口径.合计.环比`
- 季度累计 = `{季度}季度-B标.总计.总GMV`
- 季度预算 = `{季度}季度-B标.总计.目标`
- 达成率 = `{季度}季度-B标.总计.进度%`

### 其中：拉新
- 本周实际 = `周营收-财务口径.合计.拉新GMV`
- 上周实际 = 上周Excel.`周营收-财务口径.合计.拉新GMV`
- 环比 = 本周实际 ÷ 上周实际 - 1
- 季度累计 = `{季度}季度-B标.总计.拉新QTD`
- 季度预算 = `{季度}季度-B标.总计.预算`
- 达成率 = `{季度}季度-B标.总计.拉新进度`

### 其中：续报
- 本周实际 = `周营收-财务口径.合计.续报GMV`
- 上周实际 = 上周Excel.`周营收-财务口径.合计.续报GMV`
- 环比 = 本周实际 ÷ 上周实际 - 1
- 季度累计 = `{季度}季度-B标.总计.续费QTD`
- 季度预算 = `{季度}季度-B标.总计.预算`
- 达成率 = `{季度}季度-B标.总计.续报进度`

### 市场SMROI
- 本周实际 = `周营收-团队口径.合计.市场SMROI`
- 上周实际 = 上周Excel.`周营收-团队口径.合计.市场SMROI`
- 环比 = 本周实际 ÷ 上周实际 - 1
- 季度累计 = `{季度}累计同比.LCT整体.2026 SMROI`
- 季度预算 = `{季度}累计同比.LCT整体.2025 SMROI`（作为基准）
- 达成率 = 季度累计 ÷ 季度预算

### 项目SMROI
- 本周实际 = `周营收-财务口径.合计.SMROI`
- 上周实际 = 上周Excel.`周营收-财务口径.合计.SMROI`
- 环比 = 本周实际 ÷ 上周实际 - 1
- 季度累计 = `{季度}累计同比.LCT整体.2026 SMROI`
- 季度预算 = `{季度}季度-B标.总计.SMROI目标`
- 达成率 = 季度累计 ÷ 季度预算

## 三、分产品线表格数据映射

- **产品线名称**：使用层级格式（升学/素养 + 缩进项目组）
- **本周GMV（万）**：`周营收-团队口径.{项目组}.总营收`
- **环比**：`周营收-团队口径.{项目组}.环比`
- **季度累计（万）**：`{季度}季度-B标.{项目组}.总GMV`
- **季度预算达成率**：`{季度}季度-B标.{项目组}.进度%`
- **上年同期（万）**：`{季度}累计同比.{项目组}.2025 GMV`
- **季度同比**：`{季度}累计同比.{项目组}.同比增长`
- **备注**：AI自动生成（基于数据分析，简短说明关键发现）

## 四、本周速览数据源补充

- **GMV**：财务口径合计行总营收
- **拉新GMV**：财务口径合计行拉新GMV
- **续报GMV**：财务口径合计行续报GMV
- **市场SMROI**：团队口径合计行市场SMROI
- **项目SMROI**：财务口径合计行SMROI

### 分产品线数据源
- **升学业务线**：财务口径升学类项目组汇总
- **素养业务线**：财务口径素养类项目组汇总
- **同比数据**：`{季度}累计同比`Sheet
