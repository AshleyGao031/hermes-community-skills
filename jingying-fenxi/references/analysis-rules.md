## 一、分析模块

### 模块1：单周营收趋势（财务口径）

数据来源：`周营收-财务口径` Sheet 合计行（详见 `references/data-source-mapping.md`）

1. 提取总营收、拉新GMV、续报GMV、SMROI 及各自环比
2. 判断整体趋势（量增效增 / 量增效降 / 量跌效升 / 量跌效跌）
3. 提取拉新GMV Top3 项目组，分析其 SMROI 环比

### 模块2：素养项目组周营收（团队口径）

数据来源：`周营收-团队口径` Sheet 中素养类项目组（详见 `references/data-source-mapping.md`）

1. 逐项列出素养项目组：市场GMV、辅导GMV、市场SMROI 及环比
2. 汇总素养整体数据（各素养项目组求和，环比通过上周值反算）
3. 重点识别：SMROI<1、市场GMV大幅下跌、投放归零

### 模块3：各项目组GMV进度及SMROI达标（B标为主）

数据来源：`Q1季度-B标` + `当月-B标` Sheet（详见 `references/data-source-mapping.md`）

1. 逐项对比 Q1 季度总GMV进度 vs 时间进度
2. 逐项对比当月总GMV进度 vs 时间进度
3. 逐项对比拉新SMROI实际值 vs 目标
4. 按三档分类：✅优异 / ⚠️关注 / 🔴高风险

### 模块4：与上周对比（复利模块）

**仅在画像卡存在时执行。**

逐项对比画像卡中记录的上周状态：

1. 风险等级变化：是否有项目组从 ✅ 变为 ⚠️ 或 🔴？
2. 趋势方向变化：📈上升 / ➡️平稳 / 📉下降
3. 待跟踪问题跟进：上周标记的问题本周是否改善/恶化？
4. 连续周数更新：同一风险持续则 +1

**状态变化标注**：

| 标注 | 含义 |
|------|------|
| 🆕 | 本周新出现的风险 |
| ❗ | 持续存在的风险（连续2周） |
| ❗❗ | 持续≥3周，自动升级优先级 |
| ✅ | 上周有风险，本周已解除 |
| 📈 | 风险升级（⚠️→🔴） |
| 📉 | 风险降级（🔴→⚠️） |

## 二、风险识别规则

按以下规则扫描所有项目组，标记风险：

1. SMROI < 1（投放亏损）
2. SMROI实际值 < 目标值的 70%
3. 季度B标进度 < 理论时间进度的 80%
4. GMV 出现负值
5. 单周环比跌幅 > 50%
6. SMROI 很低但仍持续投放
7. **复利升级**：同一风险连续 ≥3 周 → 自动升级优先级
8. **复利降级**：风险指标连续 2 周回到安全区间 → 自动降级

### 风险升降级详细规则

**自动升级**（灵敏触发）：
- 同一风险连续 ≥3 周 → severity 从 ⚠️ 升级为 🔴
- 标注格式：`❗❗ [风险升级] {项目组} — {风险描述}，已持续{N}周`

**自动降级**（稳定确认）：
- 连续 2 周回到安全区间 → 🔴 降级为 ⚠️
- 再持续 1 周安全 → ⚠️ 降级为 ✅解除
- 标注格式：`📉 [风险降级] {项目组} — 连续{N}周恢复`

**安全区间定义**：
- SMROI ≥ 1
- SMROI实际 ≥ 目标的 70%
- 季度进度 ≥ 时间进度的 80%
- GMV 为正值
- 单周环比跌幅 ≤ 50%

## 三、画像卡更新规则

分析完成后**必须自动更新** `项目组画像卡.md`。

### 3.1 更新每个项目组的状态

- **当前风险等级**：根据本周分析结果重新评估（含自动升降级）
- **趋势方向**：与上周对比判断 📈/➡️/📉
- **连续状态周数**：风险持续则+1，状态变化则重置为1
- **滚动趋势数据**：追加本周关键指标（总营收、拉新GMV、SMROI），保留最近 **8周** 窗口，超出部分 FIFO 淘汰
- **Q1-B标进度**、**当月-B标进度**、**拉新SMROI**：更新为本周数据
- **本周摘要**：替换为本周分析发现
- **待跟踪问题**：保留未解决 + 新增本周发现 + 移除已解决

### 3.2 更新风险跟踪看板

| 情况 | 操作 |
|------|------|
| 新风险 | 新增行，首次标记=本周，持续周数=1，状态=🆕新增 |
| 持续风险 | 持续周数+1，状态=❗持续 |
| 持续≥3周 | 状态=❗❗升级 |
| 连续2周安全 | 状态=📉降级 |
| 连续3周安全 | 状态=✅解除（移出看板） |
| 本周突然解除 | 状态=⏸️观察（保留1周确认） |

### 3.3 更新记录

在画像卡顶部的更新记录表中追加本周条目。

## 四、注意事项

- **季节性因素**：参考提示词模板中业务背景的季节性规律（如线下店淡季、春节影响），避免过度报警
- **负值处理**：GMV 出现负值时标记为异常，但不假设原因，建议用户排查
- **数据积累提示**：滚动趋势数据 ≥3 周后，复利分析更有意义
- **周报模板位置**：模块5追加到分析报告末尾，作为独立章节

## 五、分析视角规则（重要！）

> 以下规则防止AI做出"数据正确但结论错误"的分析。

### 规则1：目标以考核周期整体为准
- 判断达成看考核周期整体（季度/月度），不把子周期进度问题上升为整体风险
- ❌ 错误：周度下滑 → "月度达成有风险"（当月已超额时）
- ✅ 正确：月度已达成 → 分析重点转向结构性问题和趋势

### 规则2：数据截止≠周期末
- 数据未满周期时，达成率自然偏低，不应作为风险判定依据
- 标注数据截止日，进度率仅供参考

### 规则3：分析重点随结论层次变化
- 目标已达成 → 重点转向结构性问题 + 下周期预算建议
- 目标未达成 → 才需分析缺口、追赶路径、风险等级

### 规则4：按影响度分配分析权重
- 大业务线详细分析驱动因素，小线合并带过

### 规则5：同比要看基数，不要被百分比迷惑
- 低基数效应导致高同比，用绝对增量衡量实际贡献

### 规则6：达成度异常先问预算，再问业务
- 多条线同时严重miss → 更可能是预算方法论问题，而非执行不力
- 先判断是执行偏差还是预算偏差

### 规则7：环比波动要考虑日历因素
- 连续下跌可能是节假日/工作日天数导致，不一定是真趋势

### 规则8：渠道质量比渠道规模更重要
- 高GMV渠道如果伴随高成本和低留存，是"虚胖增长"
- 追问：渠道结构偏移是主动选择还是被动结果？
### 规则9：亮点/风险要看绝对贡献，不只看达成率
- 主力线（占比>20%）达成>90%且同比正增长 = 基本盘稳固（亮点）
- 小线超额达成是锦上添花，不是核心亮点

## 六、分析表达风格规范（Ashley偏好，2026-05-07确立）

> 这是Ashley确认最喜欢的报告风格，所有分析规则以此为准。

### 核心态度：冷静客观，不被大数字带偏
面对"总营收环比暴涨890%"这种数字，**第一反应是找原因，而不是庆祝**。分析师的职责是帮老板看到真相，不是报喜不报忧。

### 表达规范

**1. 趋势结论前置**
- 每一段开头先给定性结论（量增效增/量跌效降等）
- 结论用**加粗**，让读者3秒抓住重点
- 后面跟定量数据支撑

**2. 主动纠偏，不等老板质疑**
- 预判老板看到某个数字会误读 → **提前在Attention框里解释清楚**
- 大数字拆解：续报GMV单周爆发 → 说明是季度效应还是真实增长
- 识别"假增长"：总营收涨但拉新SMROI跌，说明是存量续报而非新增质量改善

**3. 结构拆解是分析的灵魂**
- 总营收 = 拉新GMV + 续报GMV，**分开看才真实**
- 续报爆发要看：这是季节性脉冲（季度末确认），还是结构性好转？
- 拉新GMV稳定增长 + SMROI改善 = 真正的运营健康

**4. Attention框——防误读补丁**
格式：
```
> ⚠️ **Attention**
> 本周总营收的890%暴涨，主因是Q1季度合同在本周集中确认收入，
> 属于**季度效应**，不代表运营层面的质变。
> 真正反映运营健康度的指标是：拉新GMV（+25.1%）和SMROI（+27.9%）。
```
- 位置：在大数字出现的模块，紧跟数据表格之后
- 触发条件：单周环比>100%、或数字异常高/低、或可能产生误读时

**5. 结尾给行动指南，不报完数就结束**
每个模块的分析结尾：
- 要有"下周重点观察"（1-2条）
- 要有"需要业务方确认"的问题（标注[待BP确认]）
- 要有"如果继续恶化/改善，触发什么行动"的预判

**6. 趋势预测——做预期管理**
- 异常高点：说明"不预示下周会延续"，防止管理层产生不切实际的考核预期
- 异常低点：说明"可能是节假日/工作日因素，需要看连续2-3周判断趋势"

**7. 一句话总结原则**
每个模块末尾有一句话概括本质：
- ❌ "本周数据如下..."
- ✅ "本周增长主因季度末续报确认，拉新质量改善才是真正亮点"

### 参考模板片段（Ashley认可的分析风格）

**整体趋势写法：**
```
**整体趋势：量增效增，基本面健康**

本周总营收环比暴涨890%，但核心驱动力是**季度续报确认的季度效应**，**不代表运营层面有质变**。真正反映运营健康度的是：
- 拉新GMV：2,849.9万（环比+25.1%），绝对增量+572.2万 → **运营面稳健增长**
- 拉新SMROI：1.56（环比+27.9%），从1.22提升至1.56 → **投放效率持续改善**

> ⚠️ **Attention**
> 本周续报GMV从上周39.3万飙升至26,593.2万，
> 主因是Q1度假班合同在Q2首周集中确认收入，属于季节性规律，
> 不预示下周会延续此水平。
```
